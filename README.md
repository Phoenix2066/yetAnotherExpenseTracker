# yetAnotherExpenseTracker
An expense tracker in yetAnother project series. 
Built to learn

Gives a simple UI to enter the incomes and expenses with a dashboard to get insights.
Since it is a frontend project, the user data is not persistent i.e, the progress or data is lost when refreshed or closed.

## About

This project is a **Expense/Transaction Tracker** built with React and Tailwind CSS. It include fetures like adding transaction(incomes or expenses) with multiple information fields. It also provides graphs and info cards to get status or immediate info about the transaction history( Elements present but functioning not yet added).
<br>Built to learn React + Tailwind and data flow between parent and child elements.

---

## Features ##

- View summaries info of a user's transactions.
- Various field for user's ease of tranasac recording.
- A little hover interactivity

## Installation

1. **Clone the repo**

```bash
git clone https://github.com/Phoenix2066/yetAnotherExpenseTracker.git
```
2. **Install dependencies**

```bash
cd yetAnotherExpenseTracker
npm install
```

3. **Run the project**
```bash
npm run dev
```
The app should now be running at http://localhost:3000/

---

## Usage

1. Click on the the add transaction to add a new trasaction. The changes will be reflected in All Transactions tab.
2. Gives simple ui for all the transactions
